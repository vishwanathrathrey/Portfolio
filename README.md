# Data Analysis & ML Projects Portfolio

This repository contains 6 data analysis and machine learning projects.

## Projects

- **Car Price Prediction** - ML project for car price estimation
- **File Handling** - Python file handling and CSV processing
- **Flight Data Analysis** - EDA on flight records  
- **Lung Cancer Analysis** - Patient health data analysis
- **NYC Taxi Analysis** - New York taxi trip EDA
- **Traffic Data ETL** - Traffic collision data ETL pipeline

Each project folder contains:
- Jupyter notebook with analysis
- Sample dataset (data/sample/)
- README with project details
- .gitignore for proper data management

## Quick Start

Navigate to any project folder and open the .ipynb notebook in Jupyter:
