# File Handling Mini Project

## Project Overview
This mini project demonstrates basic file handling operations and simple CSV data processing in Python.

## Files
- `File_Handling.ipynb`: Notebook containing file handling exercises and workflow.
- `sales_data.csv`: Small sample dataset used for reading and processing examples.

## What This Project Covers
- Reading and writing files
- CSV parsing and basic transformations
- Handling common file operation patterns in Python

## How to Run
1. Create and activate a Python environment.
2. Install core libraries: pandas, jupyter.
3. Open `File_Handling.ipynb` and run all cells.

## Notes
- This is a practice-focused mini assignment.
- Keep examples simple and well commented for readability.

## Data Availability
- This public project includes only sample data for demonstration and reproducibility.
- The full dataset is proprietary and is not shared in this repository.
- To run with private full data, place the original files in `data/raw/` and keep lightweight samples in `data/sample/`.
