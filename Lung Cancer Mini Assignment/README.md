# Lung Cancer Mini Assignment

## Project Overview
This project analyzes lung cancer patient data to understand health and treatment patterns and derive basic insights.

## Files
- `Lung_Cancer_Mini_Assignment_1.ipynb`: Main notebook for analysis.
- `Lung Cancer Patient Health and Treatment Records/Lung Cancer.csv`: Source dataset.

## What This Project Covers
- Dataset inspection and preprocessing
- Exploratory analysis of patient attributes
- Insight generation from health and treatment records

## How to Run
1. Create and activate a Python environment.
2. Install core libraries: pandas, numpy, matplotlib, seaborn, jupyter.
3. Open `Lung_Cancer_Mini_Assignment_1.ipynb` and run all cells.

## Notes
- The dataset file is large for standard GitHub workflows.
- For portfolio uploads, keep a sample in repo and document how to access full data.

## Data Availability
- This public project includes only sample data for demonstration and reproducibility.
- The full dataset is proprietary and is not shared in this repository.
- To run with private full data, place the original files in `data/raw/` and keep lightweight samples in `data/sample/`.
