# Academic Projects (MSc Coursework)

This directory contains projects completed as part of my MSc in Machine Learning & Artificial Intelligence. These projects demonstrate my foundational understanding of core machine learning concepts, data analysis, and modeling techniques.

Each project folder contains a dedicated `README.md` with more details about the specific problem, methods used, and key outcomes.

## Projects

*   **[Car Price Prediction](./Car%20Price/)**: A regression model to estimate used car prices.
*   **[CardioRisk Prediction](./CardioRisk/)**: A classification model to predict cardiovascular disease risk.
*   **[File Handling](./File%20Handling/)**: Examples of file I/O operations in Python.
*   **[Flight Data Analysis](./Flight%20Mini%20assignmrnt/)**: Exploratory data analysis of flight records.
*   **[Lung Cancer Analysis](./Lung%20Cancer%20Mini%20Assignment/)**: Data analysis to identify lung cancer risk factors.
*   **[NYC Taxi Analysis](./NYC%20Taxi%20Analysis/)**: EDA on NYC taxi trip data.
*   **[Traffic Data Analysis](./Traffic%20Data%20Analysis/)**: ETL and analysis of traffic collision data.
