# Python File Handling Examples

## Project Overview

This academic project demonstrates basic file handling operations and simple CSV data processing in Python. It was completed as part of my MSc coursework to practice fundamental data manipulation skills.

## Key Concepts Learned

- **File I/O**: Reading from and writing to text files using Python's built-in functions.
- **CSV Data Processing**: Using the `pandas` library to read, parse, and perform basic transformations on CSV data.
- **Jupyter Notebook**: Utilizing notebooks for interactive code execution and documentation.

## Technologies Used

- Python
- Pandas
- Jupyter Notebook

## How to Run

1.  Ensure you have Python, Pandas, and Jupyter installed.
2.  Open the `File_Handling.ipynb` notebook in a Jupyter environment.
3.  Execute the cells sequentially to see the code examples and their output.
