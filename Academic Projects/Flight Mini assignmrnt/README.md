# Flight Data Mini Assignment

## Project Overview
This project performs exploratory analysis on flight records to identify trends, delays, and route-level patterns.

## Files
- `Flight_Data_Mini_Assignment_2.ipynb`: Main analysis notebook.
- `Flight Dataset - CSV(in).csv`: Flight dataset used for analysis.

## What This Project Covers
- Data loading and cleaning
- EDA for time, route, and performance patterns
- Basic analytical insights from flight operations data

## How to Run
1. Create and activate a Python environment.
2. Install core libraries: pandas, numpy, matplotlib, seaborn, jupyter.
3. Open `Flight_Data_Mini_Assignment_2.ipynb` and run all cells.

## Notes
- If this dataset grows large, consider storing only a sample in GitHub and linking full data externally.

## Data Availability
- This public project includes only sample data for demonstration and reproducibility.
- The full dataset is proprietary and is not shared in this repository.
- To run with private full data, place the original files in `data/raw/` and keep lightweight samples in `data/sample/`.
