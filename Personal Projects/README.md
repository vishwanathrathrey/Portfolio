# Personal Projects

This directory contains personal projects built to demonstrate practical, end-to-end implementation of data engineering, MLOps, and Generative AI systems. These projects showcase my ability to design, build, and deploy real-world applications.

Each project folder contains a dedicated `README.md` with detailed information on the architecture, setup, and usage.

## Projects

*   **[AI Code Review](./AI%20Code%20Review/)**: An automated code review system using a Large Language Model (LLM) to analyze GitHub pull requests.
*   **[Credit Risk ML System](./credit-risk-ml-system/)**: An end-to-end machine learning system for credit risk prediction, featuring a PySpark ETL pipeline and a FastAPI-powered API.
*   **[Customer Data Pipeline](./customer-data-pipeline/)**: A containerized ETL pipeline for ingesting and processing customer data.
