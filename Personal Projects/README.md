# Personal Projects

## Overview

This directory contains personal projects focused on applying software engineering principles to Machine Learning, Data Engineering, and Generative AI problems. These projects demonstrate my ability to build production-style, end-to-end systems that are robust, scalable, and solve practical problems. They mark the transition from academic concepts to real-world application.

## Engineering Capabilities Demonstrated

These projects showcase experience in:

*   **End-to-End ML & AI Systems:** Building complete solutions from data ingestion and processing to model training, deployment, and serving via APIs.
*   **API Design & Development:** Designing and implementing RESTful APIs using FastAPI and Flask to serve models and data, a core component of modern ML systems.
*   **Containerization & MLOps:** Using Docker and Docker Compose to create reproducible, isolated environments for development and deployment, a foundational MLOps practice.
*   **ETL & Data Engineering Pipelines:** Implementing data ingestion and transformation pipelines to process raw data into a clean, usable format for analytics and model training.
*   **LLM Integration:** Integrating Large Language Models (LLMs) into software workflows to perform complex, human-like tasks such as code analysis and review.
*   **Scalable System Architecture:** Designing modular, microservices-based architectures that allow for independent scaling and maintenance of components.
*   **Database Integration:** Storing and managing data in relational databases like PostgreSQL within a data pipeline context.
